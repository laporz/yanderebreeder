# CLAUDE.md

## Project Overview

このプロジェクトは、Godotで制作するシーン切り替え型ADVゲームです。  
開発にはClaude Codeを使用します。

## Setting Rules

- ヒロインは2000歳を超える成人済みのエルフである
- ヒロインは人間界に頼れる相手が少ないが、自立した意思を持っている
- 主人公とヒロインは保護・被保護の関係ではない
- 屋敷への滞在はヒロイン自身の意思によるもの
- 恋愛・親密イベントは対等な成人同士の関係として扱う

## Development Stack

- Engine: Godot
- Genre: シーン切り替え型ADV
- Development Assistant: Claude Code

## Development Rules

- 既存ファイルを確認してから変更する
- 仕様変更時は関連ドキュメントも更新する
- シーン、UI、状態管理、シナリオ表示ロジックを分離する
- Godotのシーン構成とスクリプト責務を明確にする
