# Scenario Rules

## 基本文体

- ADV形式の地の文と会話文で構成する
- キャラクターの感情変化を丁寧に描写する
- 数値変化は必要に応じてUI側で表示する
- 会話は好感度段階に応じて距離感を変える
- ファンタジー世界観に合わせた語彙を使う

## 好感度別トーン

### 0〜20: 初対面でよそよそしい

- 警戒している
- 返答が短い
- 距離感がある

### 21〜40: 慣れてきた

- 少し会話が続く
- 主人公の屋敷や人間界に興味を示す
- 反応が柔らかくなる

### 41〜60: 仲良し

- 雑談や冗談が増える
- 主人公を信頼しはじめる
- 日常的な交流が自然になる

### 61〜80: 気になる人

- 特別な感情が見え始める
- 照れや迷いが増える
- 距離感が近くなる

### 81〜100: ラブラブ

- 好意が明確になる
- 親密な会話が中心になる
- 対等な成人同士の関係として描写する

## 設定上の注意

- ヒロインは2000歳を超える成人済みのエルフ
- ヒロインは自立した意思を持っている
- 主人公とヒロインは保護・被保護の関係ではない
- 屋敷への滞在はヒロイン自身の意思によるもの
- 恋愛・親密イベントは対等な成人同士の関係として扱う
