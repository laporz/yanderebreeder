Game Spec

概要

Godotで制作するシーン切り替え型ADVゲーム。

開発にはClaude Codeを使用する。

ヒロインとのやりとりを通じて、エンディングが3種類に分岐する。

Development Stack

EngineGodot
Genreシーン切り替え型ADV
Development AssistantClaude Code

基本ループ

  1. MorningSceneで会話、ふれあい、ステータス確認を行う。
  2. 何らかの数値変動、または会話の失敗があった場合、次のシーンへ遷移する。
  3. NightSceneで買い物、食事、会話、ふれあい、就寝を行う。
  4. 条件を満たすとMidnightSceneへ遷移する。
  5. 就寝後、翌日またはエンディング判定へ進む。

エンディング

ヒロインとのやりとりによって、エンディングが3種類に分岐する。

具体的な分岐条件は未定義。今後、好感度・依存度・日数・購入済みアイテムなどを使って設計する。
